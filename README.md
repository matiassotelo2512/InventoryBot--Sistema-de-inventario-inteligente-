# InventoryBot--Sistema-de-inventario-inteligente-
Sistema de gestión de inventario para el equipo, diseñado para una entrada de datos rápida mediante Telegram y una visualización centralizada a través de un Dashboard Web.

Descripción
Sistema de gestión de inventario para el equipo, diseñado para una entrada de datos rápida mediante Telegram y una visualización centralizada a través de un Dashboard Web.

Stack Tecnológico
Backend: FastAPI (Python)

Base de Datos: PostgreSQL

Interfaz de Entrada: Telegram Bot API

Interfaz de Visualización: Streamlit

Gestión de Entorno: Poetry

Contenedores: Docker & Docker Compose

Estructura del Proyecto
Plaintext
/api            # Endpoints de la API central
/bot            # Lógica del bot de Telegram
/dashboard      # Dashboard de Streamlit
/core           # Configuración compartida y modelos
/tests          # Pruebas unitarias
